Caso 007: “O Hacker Silencioso”

Um movimento estranho foi detectado no sistema às 3h da manhã. Um usuário acessou a rede com um nome criptografado. Cabe a você identificar se ele é confiável ou um possível invasor.

  • Se for 5 ou maior, permita o acesso.
  • Se for 0 e maior que 10, imprima uma mensagem de alerta: "Invasor detectado!".
  • Para os demais casos, negue o acesso.

Imprima uma mensagem adequada para cada caso.
