🕵️ Caso 5: O Mistério da Identidade Dupla

🧠 Hipótese Inicial

Suspeito 1 e Suspeito 2 parecem iguais. Mas... serão realmente a mesma pessoa?

🔬 Área de Investigação

let suspeito1 = 42;
let suspeito2 = "42";

console.log(suspeito1 == suspeito2); // Resultado?
console.log(suspeito1 === suspeito2); // Resultado?

📌 Pistas

  • Existem dois tipos de igualdade em JavaScript.
  • Tipos diferentes podem se comportar de maneiras... suspeitas.
  • Um deles é mais rigoroso que o outro.
